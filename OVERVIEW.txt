✅ 8 TABELAS PRINCIPAIS

   ┌─────────────┐
   │  profiles   │ ← id (FK → auth.users)
   │  counters   │   username, email, full_name
   │  timestamps │   avatar_url, bio
   └─────────────┘   followers_count, following_count, posts_count
         │
         ├─→ ┌──────────┐
         │   │  posts   │ ← id (PK), user_id (FK)
         │   │ counters │   title, description, video_url, thumbnail_url
         │   │timestamp │   likes_count, comments_count
         │   └──────────┘
         │        │
         │        ├─→ ┌────────┐
         │        │   │ likes  │ ← id, user_id (FK), post_id (FK)
         │        │   │UNIQUE  │   created_at
         │        │   └────────┘   (triggers: auto increment counter)
         │        │
         │        └─→ ┌──────────┐
         │            │comments  │ ← id, user_id (FK), post_id (FK)
         │            │ counter  │   content, likes_count
         │            │timestamp │   (triggers: auto increment counter)
         │            └──────────┘
         │
         ├─→ ┌────────┐
         │   │follows │ ← id, follower_id (FK), following_id (FK)
         │   │  CHECK │   follower_id ≠ following_id (no self-follow)
         │   │UNIQUE  │   (triggers: auto update counters)
         │   └────────┘
         │
         └─→ ┌──────────────────────┐
             │conversations &       │
             │conversation_         │ ← Direct messages
             │participants          │   (Realtime enabled)
             │                      │   messages table
             └──────────────────────┘
